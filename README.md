# Telegram Booking Bot (Template)

Готовий Telegram-бот для запису клієнтів:
- вибір послуги
- вибір дати і часу
- показ тільки вільних слотів
- збереження в SQLite
- повідомлення адміну про новий запис

## Швидкий запуск (10–15 хв)

### 1) Створіть бота і отримайте TOKEN
Через @BotFather у Telegram.

### 2) Створіть .env
Створіть файл `.env` у корені проєкту і вставте:

TOKEN=YOUR_BOT_TOKEN
ADMIN_IDS=123456789

> ADMIN_IDS може бути кілька через кому: `123,456`

### 3) Встановіть залежності
```bash
pip install -r requirements.txt
